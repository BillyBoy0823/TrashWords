# TrashWords
e107 v2.x Plugin - ADMIN - Just Click &amp; Drag Bad words to the Trash (Profanity Filter)!

TrashWords allows ADMIN to simply double-click BAD words, drag them to the textbox and save them to the Profanities filter. The Profanities filter must be enabled in Admin>Settings>Preferences>Content Filters. If a word was added by mistake, that is also where ADMIN can go to remove the mistakes from the filter.

Hope you find this easy and useful.
